# **App Name**: AITAM Connect

## Core Features:

- Enquiry Submission: Allow users to submit enquiries with a rich text editor for detailed questions.
- Q&A Display: Display questions and answers in an organized format on the user dashboard, updated in real-time.
- Automated Archiving: Archive questions older than 5 days to the user's inbox, storing data for 30 days before permanent deletion.
- College Domain Authorization: Authorize user access using the college domain (adityatekkali.edu.in) for secure registration and login.
- AI-Powered Answer Summarization Tool: Use generative AI to analyze user answers, suggest grammar improvements, and identify common, informative responses. A tool where AI identifies trends and verifies with an administrator for final approval, then summarizes the most comprehensive responses.
- Administrator Moderation: Route proposed answer summaries through administrator review and action before displaying to the user.
- Profile and Customization: Enable users to upload a profile picture and showcase the username prominently on the home screen. Provide unique login options including Google, email, and phone, and allow users to create new accounts (admin, student, alumini).
- PPT Maker Integration: Integrate a PPT maker for creating presentations within the platform.
- Resume Builder Integration: Integrate a resume builder that suggests recommended courses and tips to crack job interviews, tailored to company requirements and latest resume models.
- Latest News and Information: Display latest news and information from (https://adityatekkali.edu.in/autonomous.php) on the home screen.

## Style Guidelines:

- Primary color: Vibrant blue (#29ABE2) to evoke a sense of trust and knowledge.
- Background color: Colorful background gradients to attract students.
- Accent color: Yellow (#FFD700) for highlighting important information and CTAs, providing a sense of importance and action.
- Body and headline font: 'PT Sans', a humanist sans-serif that provides a modern and welcoming look.
- Use a set of simple, clear icons to represent different categories of questions and actions. Integrate the AITAM logo from (https://adityatekkali.edu.in/assets/images/logo/logo3.webp).
- Showcase username, and display latest news and information (summarized by AI where possible) from (https://adityatekkali.edu.in/autonomous.php) on the home screen.
- Create Testimonials Component by Adding Mock Data with Student names, quotes, and roles styled closely using Tailwind CSS into the home page.
- Use animated content and sliding elements for a modern user experience.
- Implement stylish scroll bars for enhanced visual appeal.
